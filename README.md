# All My Books

This is the backend part of All My Books app.

If you want to work with this repo you need the following variables:

* PORT=3000
* ENV=development
* DB= MongoDB
* SECRET
* GOOGLE_ID
* GOOGLE_SECRET
* GOOGLE_CALLBACK
* FACEBOOK_ID
* FACEBOOK_SECRET
* FACEBOOK_CALLBACK
* FRONTENDPOINT


Here is the [frontend](https://github.com/gbrenesm/All_My_Books_client)
